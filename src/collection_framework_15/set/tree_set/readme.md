### TreeSet

> how to use

```java
TreeSet<E> treeSet = new TreeSet<E>();
// example
TreeSet<String> treeSet = new TreeSet<String>();
```

> Method

| return type | method | description      |
|-------------|--------|------------------|
| E           | first()| 제일 낮은 객체를 리턴 |
| E           | last() | 제일 높은 객체를 리턴|
