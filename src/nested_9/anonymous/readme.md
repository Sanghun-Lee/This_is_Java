### 익명 객체

익명 객체란 이름이 없는 객체를 말한다.

익명 객체는 단독으로 생성할 수 없고, 클래스를 상속하거나 인터페이스를 구현해야만 생성할 수 있다.

익명 객체는 필드의 초기값이나 로컬 변수의 초기값, 매개 변수의 매개값으로 주로 대입된다.

| 9.5.3 익명 객체의 로컬 변수 사용 | local_class  |
