## JavaFX

JavaFx는 java7부터 JDK에 포함되어 있다. 하지만, Java11부터 다시 JDK에서 제외되었다.

UI는

`AWT` 에서 `Swing`을 거쳐서 `JavaFx`로 오게 되었다.

JavFx의 구성요소로는 아래와 같다.

> 레이아웃 (자바 코드 파일 또는 FXML)
>
> 외관 및 스타일 (CSS)
>
> 비즈니스 로직 (자바 코드 파일)
>
> 리소스 (그림파일, 동영상 파일....)
